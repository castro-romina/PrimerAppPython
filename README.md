# PrimerAppPython 

Aplicación de gestión de alumnos desarrollada en Python con interfaz gráfica (Tkinter).

## Descripción

Permite administrar alumnos y sus cursos mediante una ventana de escritorio, mostrando la información en consola.

## Funcionalidades

- Login con usuario y contraseña
- Agregar alumnos con su cantidad de cursos
- Ver la lista completa de alumnos
- Consultar la cantidad de cursos de un alumno
- Salir de la aplicación

## Tecnologías

- Python 3
- Tkinter

## Cómo ejecutar

```bash
python app.py
```
